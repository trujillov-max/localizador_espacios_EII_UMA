# Localizador EII-UMA — móvil con zoom por pinza

Cambios:
- scroll táctil normal con un dedo;
- zoom del mapa con pinza de dos dedos;
- botón de centrar como símbolo `◎` a la derecha de Resultados;
- sin botones de zoom visibles en móvil;
- controles de zoom conservados en escritorio.

Sustituye el `index.html` de GitHub por este archivo.
