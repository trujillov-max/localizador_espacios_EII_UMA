# Localizador de Espacios EII-UMA

Aplicación web estática para localizar espacios de la Escuela de Ingenierías Industriales de la Universidad de Málaga.

## Publicación rápida en GitHub Pages

1. Crea un repositorio público llamado `localizador-eii-uma`.
2. Sube estos archivos al repositorio.
3. Ve a Settings > Pages.
4. En Source, elige Deploy from a branch.
5. Selecciona `main` y `/root`.
6. Guarda los cambios.
7. La web se publicará en `https://TU_USUARIO.github.io/localizador-eii-uma/`.
